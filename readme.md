## Requirements
- PHP >= 8.3
- Nette Framework
- Composer
- Bootstrap
- Naja
- Node.js
- Drago Project core packages

## Installation
Warning, it overwrites files in the project! which are listed in the composer in the replace section.
```bash
composer require drago-ex/project-auth
```
